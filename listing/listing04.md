Что выведет программа? Объяснить вывод программы.

```go
package main

func main() {
	ch := make(chan int)
	go func() {
		for i := 0; i < 10; i++ {
			ch <- i
		}
	}()
	for n := range ch {
		println(n)
	}
}
```

Ответ:
```
Программа выведет значение от 0-9, а потом fatal erorr - deadblock.
Произойдет это в результате того, что цикл for range channel считывает значения из канала, пока он не закроется. В нашей программе мы не закрываем канал, т.е. цикл после считывания значения 9, будет ждать очередного значения, в результате чего произойдет deadblock, т.к. значение туда никогда не придет.
```
