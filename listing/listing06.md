Что выведет программа? Объяснить вывод программы. Рассказать про внутреннее устройство слайсов и что происходит при передачи их в качестве аргументов функции.

```go
package main

import (
	"fmt"
)

func main() {
	var s = []string{"1", "2", "3"}
	modifySlice(s)
	fmt.Println(s)
}

func modifySlice(i []string) {
	i[0] = "3"
	i = append(i, "4")
	i[1] = "5"
	i = append(i, "6")
}
```

Ответ:
```
Программа выведет:
[3,2,3]
Внутреннее устройство слайса:
type slice struct {
	array unsafe.Pointer
	len   int
	cap   int
}, где array - указатель на массив, len — текущая длина среза, cap — длина внутреннего массива (емкость).
Программа выведет именно [3,2,3] потому что в функцию передается слайс, у которого len = 3, cap = 3, в строке i[0] = "3" действительно поменяется 0-ой элемент слайса, т.к. указатель s и i указывает на тот же самый базовый массив. 
В строке i = append(i, "4") переменная i начнет указывать уже на новый аллоцированный массив, т.к. len=cap=3, чтобы добавить элемент необходимо увеличить базовый массив, поэтому из старого массива копируются все елементы в новый, cap увеличивается в два раза, и теперь переменная s указывает на один массив, а i на другой, теперь все изменения со слайсом i не будут влиять на слайс s.
```
